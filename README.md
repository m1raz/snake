# ASCII snake (Windows console)

I have nothing to do. So I made an ASCII snake. =_="

# compile

Use g++. I used Code::Blocks 17.12 build in MinGW.

`g++ -Wall -fexceptions -g -std=c++11 -Ixlnt/include -lxlnt -c main.cpp -o snake.exe`

# play

| Key         | Action        | 
|------------ |:-------------:| 
| UP          | move up       |
| DOWN        | move down     |
| LEFT        | move left     |
| RIGHT       | move right    |
| SPACEBAR    | add speed     |

That's all.
